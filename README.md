## 介绍
######  环境依赖
node v0.10.28+
reids ~

######  部署步骤
1. 添加系统环境变量
    export $PORTAL_VERSION="production" // production, test, dev


2. npm install  //安装node运行环境



3. 启动两个配置
    node user.js
    npm start
